# Aplica-o-web-desenvolvida-em-React-para-demonstrar-os-conceitos-do-Scrum-incluindo-Dashboard.
Scrum Manager é uma aplicação web desenvolvida em React e Vite para demonstrar os principais conceitos do framework Scrum. Possui Dashboard, Sprint Backlog, Product Backlog, métricas ágeis, gerenciamento de tarefas e visualização de papéis, eventos e artefatos do Scrum.
